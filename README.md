# Classification Assignment – Machine Learning Portfolio

## Overview

This repository contains a collection of end-to-end Machine Learning Classification projects built using an industry-style workflow.

Each project follows the complete machine learning pipeline from data preprocessing to model evaluation, with a strong emphasis on feature engineering and model interpretability.

---

## Projects Included

| Project                           | Problem                               | Algorithms                         |
| --------------------------------- | ------------------------------------- | ---------------------------------- |
| German Credit Risk                | Predict customer credit risk          | Logistic Regression, Decision Tree |
| Bank Marketing                    | Predict term deposit subscription     | Logistic Regression, Decision Tree |
| Adult Income Census               | Predict income category (>50K / ≤50K) | Logistic Regression, Decision Tree |
| Employee Attrition                | Predict employee attrition            | Logistic Regression, Decision Tree |
| Cardiovascular Disease Prediction | Predict cardiovascular disease risk   | Logistic Regression, Decision Tree |

---

## Standard Workflow Used

Each project follows the same industry-oriented workflow:

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Roll Rate Bucket Creation (where applicable)
5. Target Variable Creation
6. Weight of Evidence (WOE) Encoding
7. Information Value (IV) Analysis
8. Feature Selection
9. Variance Inflation Factor (VIF) Analysis
10. Train-Test Split
11. Logistic Regression
12. Decision Tree
13. Hyperparameter Tuning using GridSearchCV
14. Model Evaluation
15. Feature Importance
16. Decision Tree Visualization
17. Model Comparison
18. Final Conclusion
